 ***********************************************************************
 
 BadiStick est une application web Java EE qui fournit les services
 suivants :
 
 - articles sur le badminton
 - communauté de badistes (forum, chat...)
 - simulation de matches / calcul de moyennes
 - statistiques personnelles
 - recherche de joueur ou de club
 - recherche de tournoi
